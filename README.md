# Minesweeper Ultra

A minesweeper game built with svelte. Enjoy

## Tech Stack

Svelte

## Running

Run this project

```bash
  git clone https://github.com/lancerdonnie/minesweeper
  cd minesweeper
  npm install
  npm run dev
```

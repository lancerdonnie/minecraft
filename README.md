# Minesweeper Ultra

A minesweeper game built with svelte. Enjoy

## Tech Stack

Svelte

## Running

Run this project

```bash
  git clone https://github.com/lancerdonnie/minecraft
  cd minecraft
  npm install
  npm run dev
```
